TAḤANUT - Aɣerbaz amenzu Hacmi d warraw-is.
Tella-d tmeɣra n taggara n useggas i yinelmaden.
Tameddit n wass-nni n larebɛa iɛeddan, 03 yulyu 2019, temlal tasa d way turew deg uɣerbaz amenzu Hacmi d warraw-is n taddart n Tḥanut.
D tagnit ideg mlalen akk yinelmaden ɣer tama n yiselmaden-nsen ɣer tama n yiselmaden d yinemhalen iqdimen i ixedmen deg uɣerbaz-agi.
Ala acmumeḥ d lferḥ i d-yettbanen ɣef wudmaen, ma d ilsawen ur cuḥḥen ara i uqessar d usmeki n wayen sɛeddan deg umkan-agi ɛzizen fell-asen aṭas.
Mi ak-nnan d 18h00 n tmeddit, tebda tmeɣra i d-yellan i lmend n yinelmaden i d-yufraren deg yikayaden n taggara n useggas aɣurbiz.
Mbeɛd awal n tazwara, n unemhal n uɣerbaz-agi amenzu, Mass Neṣṣaḥ Amar, i d-yennan: “S lferḥ meqqren i d-nezdi tawcult-nneɣ tis snat;
inelmaden d yiselmaden-nsen akked inemhalen ixedmen dagi.
D tagnit daɣen, i usebɣes n yinelmaden-nneɣ i d-yufraren deg yikayaden-nsen”.
Ma si tama-s uselway n tɣiwant Mass Akli Caɛban, yuɣal-d ɣer ugemmuḍ igerrzen i d-yewwi uɣerbaz-agi deg ukayad n uswir wis 5, dɣa yenna-d : “Nefreḥ aṭas s ugemmuḍ n uɣerbaz-agi, tajmilt ad tuɣal i yiselmaden d unemhal d yixeddamen merra, nekkni aql-aɣ dagi deffir-sen”.
Telli tmeɣra-agi s tezlit “Taɣribt-iw” n ucennay Yidir, i d-ccnan yigerdan n tddukkla tadelsant Imnar n Tdukli, yeqqaren dɣa deg uɣerbaz-agi.
Syin akkin, yebda ufraq n warrazen ɣef yinelmaden i d-yufraren deg yal tasmilt, bdu-tt-id si uswir n uheggi ɣer wis ukkuẓ.
Tella-d diɣ tejmilt i uselmad s yiwen n usefru, i d-tewwi tlemẓit Lezzam Ḥayet, deffir-s kan, ulin-d kra n yinelmaden yeqqaren deg uɣerbaz-agi, cnan-d asiɣret “Aɣerbaz-iw”.
Tin ɣur-s, kemmlen deg ufraq n yigerdasen ɣef merra yinelmaden i d-yewwin akayad n uswir wis 5, mbeɛd, tizlit-nni “Tulawin” n Cerif Ḥamani, i d-ɣennant tullas n terbaɛt n ccna n Tddukkla Imnar n Tdukli.
Uɣalen-d ɣer tukci n warrazen i ukkuẓ n yinelmaden imenza, i d-yufraren s tezmilt igerrzen deg ukayad n uswir wis 5.
Tettunefk diɣ tegnit i ccna, ideg i d-kemmlent tullas-nni n terbaɛt n ccna tizlit-nsent taneggarut “Kečč ruḥ, nekk ad qqimeɣ” n Lewnis At Mengellat, deffir-sent kan tecna-d tlemẓit Bundawi Anya, tizlit-nni “Asaru” n Malika Dumran.
Sya ɣer da, ttilint-d tnagiyin n yiselmaden d yinemhalen ixedmen yakan deg uɣerbaz-agi, ɣer taggara, tella-d tegnit n ccḍeḥ.
Ma deg wawal n taggara n unemhal n uɣerbaz-agi, Mass Neṣṣaḥ Amar, yerra tajmilt i kra n win i d-yerran i teɣri n uɣerbaz, am wakken daɣen yesnemmer iɛeggalen n tddukkliwin-agi Imnar n Tdukli d wid n tddukkla tanaddalt n At Ɛissa Mimun (ASCA AAM), i d-yefkan afus n tallelt.
