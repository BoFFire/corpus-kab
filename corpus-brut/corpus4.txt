BGAYET - Asmekti n umulli wis 21 n tmenɣiwt n Meɛtub Lwennas.
Ayen din d leqdicat, sya u sya.
I lmend n usmekti s umulli wis 21 n tmenɣiwt n ucennay aɣewway Meɛtub Lwennas, yuget leqdic adelsan deg tmurt n leqbayel merra akked tmura tiberraniyin (Fransa d Kanada).
Ass n ttlata 25 yunyu 2019, ad zemmen 21 n yiseggasen ɣef wasmi i d aɣ-yeǧǧa yitri n ccna amaziɣ, i wumi mazal ṣṣut-is yettbeɛziq.
Dɣa dagi di tmurt, yettili-d leqdic adelsans wazal-is i lmend n usmekti-agi wis 21 deg lwilaya n Tizi Uzzu d Bgayet.
Di Ticci, d Axxam n yidles Idir Ɛacur n Bakaru, i d-yesdurin ahil n tidukla tadelsant Asaki, i d-iheggan sin wussan n usmekti, s waṭas n yiremad idelsanen.
Ass-nni n lǧemɛa, tella-d temlilit taseklant d uskasi akked umaru d umedyaz Yasin Ḥebbac, i yewwin awal ɣef umennuɣ n umedyaz d ucennay Meɛtub Lwennas.
Imeslay-d daɣen umeskar-agi ɣef udlis-is i wumi isemma “Lwennas Meɛtub, neɣ abrid ɣer umyit”, i d-yessuffeɣ ɣer tezrigin tafat.
Sendiḍelli n ssebt, wass n usmekti wis sin n tiddukkla-agi, ddima deg Uxxam n yidles Idir Ɛacur, yebda leqdic-a, ɣef 10h d tasawent, s ddeqs n temsikniyin, yerzan tudert d umecwar n ucennay aqbayli, yefkan iman-is d asfel i tmaziɣt.
Tettunefk daɣen tegnit i tmeɣra n ccna, ideg i d-ttwacnant aṭas n tezlatin-is yeččuren d iḥulfan, i mazal anza-nsent deg yimeẓẓuɣen n medden.
ɣef 14h n tmeddit, yella-d usarag d uskasi, sɣur umaru Allas Di Tlelli akked Nadya Meɛtub, d taǧǧalt n ucennay ameqqran Meɛtub.
Yezzi deg-s wawal ɣef usentel-agi: “Meɛtub Lwennas: akkin i unaẓur, d amdan bu waṭas n tsektiwin”.
Mi yessuli usarag-agi, tettunefk tegnit i usteqsi n yimsirgen-agi i d-yellan d inebgawen n lqahwa taseklant n Ticci.
Yetturaǧu, ad d-tili tmeɣra meqqren tameddit n wass-agi ɣef 19h30 n tmeddit deg Unnar Aɣurbiz n Bgayet.
D leqdic ara d yilin s lmendad n Nadiya Meɛtub d tidukla Meɛtub Lwennas i usmekti d usiweḍ, tameddit-agi deg lwilaya n Bgayet.
D tameɣra n tejmilt, ideg ara cennun ddeqs n yicennayen imeqqranen, yecban Uleḥlu, Ali Ideflawen, Malika Dumran, Zeddek Lmulud akked waṭas-nniḍen.

Llan-d kra n wuguren.
Ɣas yezri ukayad n BAC deg tegnatin igerrzen deg tuget n wammasen n usɛeddi deg twilayt n Wehran.
Maca llan-d kra n wuguren, ɣas mačči d imeqqranen, maca sneqsen almes (concentration) ɣef waṭas n yinelmaden i iḍefren timsirin-nsen akken ilaq ɣef teɣzi n useggas.
Gar wuguren-a, ad d-nebder amqelleɛ i d-yellan gar kra n yinelmaden d ucqirrew n yiselmaden i ten-iɛussen deg tneɣrit.
Yiwen seg yinelmaden-a, yessaweḍ yerẓa ddjaj n ṭṭaq n tneɣrit deg wass amenzu d wass wis kraḍ, acku iselmaden i t-iɛussen ur t-ǧǧin ara ad yexdem lebɣi-s.
Azekka-nni, yusa-d unemhal n usegmi n Wehran ɣer wammas-a, ur yessaweḍ ara ad yessers tilufa, acku kra n yinelmaden kemmlen deg leɛyaḍ.
Annect-a yeḍra-d deg wammas n Cerraṭ Ḥmed deg Ɛin Tturk.
Mi nesteqsa inelmaden-nniḍen yeqqaren d wigi iɛerḍen ad sxerben u ad xedmen ccwal, nnan-aɣ-d belli “ur ɣrin ara aseggas kamel, usan-d imir-a deg kraḍ n wussan, bɣan ad d-ḥellin akayad-a”.
D tigawin am ti i d-yessebganen aṣubbu n uswir n ulmud deg uɣerbaz Azzayri, acku ula d akayad meqqren yecban win n BAC, truḥ-as lbenna-s.
Ɣer taggara, ad d-nini daɣen ula d iselmaden begnen-d kra n lɣiḍ ɣef snat n tlufa.
Tamezwarut, ttemlilin-d uguren deg berra sɣur kra n yinelmaden.
Tis snat, d addad ur nelhi ara n tuččit i asen-d-fkan di leftur, acku tettas-d akken semmḍet, rnu tidersi n tgella i d-ttaṭṭafen.
Ɣef waya, ssutren aɛiwed n tmuɣli deg tɣawsiwin am ti, ɣas ur sɛint ara azal d ameqqran, maca ttɛawanent akken ur yettṣubbu ara uswir ugar.

Asmekti n tefsut taberkant.
Zrin 18 n yiseggasen ɣef tikli tameqqrant n Leqbayel.
14 yunyu 2001, neɣ tikli n Tefsut taberkant, d ass ideg ffɣen Yeqbayliyen deg tikli ɣer Lezzayer tamanaɣt.
D ass anda ruḥen wazal n sin neɣ tlata n yimelyan n yiqbayliyen ɣer tselwayt n tegduda i wakken ad ssersen Tiɣerɣert n Leqser i yellan d aḍris yessuturen tugdut, izerfan n umdan akked tmagit.
Acu kan, adabu ur ten-yeǧǧi ara ad awḍen iswi, imi i asen-d-ihegga deg ubrid-nsen, imcumen-is i asen-igan tawaɣit s ddaw leɛnaya n yiɣallen n laman.
D tikli ara yeqqimen deg umezruy i lebda.
Tikli-a, tamelwit i ixedmen Yeqbayliyen mgal adabu ameṣbaṭli, tuɣal d annar n umennuɣ gar yilemẓiyen d yiqeddacen n yiɣallen n laman.
Tegla-d Tefsut tabertkant s temda n yidammen, deg-s ɣerqen warrac.
Mmuten ugar n 127 n yilemẓiyen, ttwajerḥen ugar n 3000 deg-sen, mebla ma nettu wid i d-yeggran d imeεḍaṛ i lebda.
D azemz ara yeqqimen deg wul n yal Aqbayli, acku ulac tawacult taqbaylit i iselken seg leqṛiḥ n tyita, imi jeṛḥent tasiwin n tyemmatin, ttrun yibabaten idammen.
Ffɣen warrac ass-nni i wakken ad ssutren izerfan-nsen iṛuḥen, i llan ssuturen seg Tefsut n Yimaziɣen 1980, ladɣa tamsalt n tutlayt d yidles amaziɣ i yeqqlen di rrif deg tmurt n Masensen.
Mebla ma nemmeslay-d ɣef temḥeqṛanit n udabu ameṣbaṭli i yekksen i Uqbayli izerfan-is n uxeddim, n tlelli n umeslay, n yidles, n tɣuṛi, d waṭas n temsal-nniḍen.
Ayagi imi bɣan tamurt, ad tili d taεṛabt d tineslemt.
Rnu i waya, ula d tamettant n yilemẓi Masinisa Germaḥ deg At Dwala sɣuṛ yiǧadarmiyen ass n 18 yebrir 2001, terna tecεel times deg wulawen n Yiqbayliyin, i yugin ad knun i ddel, imi si zik Amaziɣ yettwassen d akken d mmi-s n tlelli.
Ass-a, 14 yunyu 2019, εeddan 18 n yiseggasen segmi faqen Yeqbayliyen s tkerkas d txubbta n udabu.
Tura ɣas ukin-d ula d wiyaḍ seg nadam-nsen, lameεna ulawen d wallaɣen, ad qqimen akked yimeɣṛaṣen n Tefsut taberkant, d Tefsut n Yimaziɣen.
TALA ƐETMAN - Lɛid tameẓyant.
Gar wansayen n zik d unecreḥ n tura.
Am wakken tuɣ tannumi taddart n Tala Ɛetman, yal aseggas zellun yimezdaɣ-is lewziɛa (timecreṭ) deg wass n lɛid.
Ihi, akken kan i d-iban d lɛid, mmlalen-d Wat taddart deg tlemmast n sin yibraḥen: Abraḥ n Tezmalt n Wadda akked win n Tezmalt n Ufella.
Yal wa yewwi-d yid-s arraw-is neɣ igerdan n twacult-is tameqqrant i d-yelsan iselsa-nsen ijdiden, am waken diɣ i d-glan n yililucen i d-uɣen ɣef demma n lɛid.
Tekker taddart ad telḥu s yisuɣan d unecreḥ n yigerdan akked zhir yililucen-nsen.
Ula d imeqqranen s umur-nsen: Ufan iman-nsen dayen kan akked yimeddukkal i d-yerzan n lebɛid;
usan-d ula seg lwilayat-nnḍen.
D ass i deg-i tettemlili tasa d wayen turew ula deg tlemmast n taddart mačči kan deg yixxamen.
Xas ulamma nnecraḥen gar-asen, meɛna ttraǧun s ccuq lawan n tmezla.
Ayen i d-yeḍran deg yiḍ, dɣa, n lɛid imi ur zmiren ara ad tt-ǧǧen alamma d azekka-nni imi d anebdu.
“Ad ten-nezlu tura u ad nheggi tisɣar imi azekka d azɣal, ma ulac ad yexser uksum”, i aɣ-d-yenna yiwen n yimḍebber.
Akken kan i d-yers uzger amezwaru i tmezliwt, ha-ten-in wwḍen-d warrac akken ma llan i wakken ad ṭṭfen tugniwen akked yiɣersiwen i iruḥen d iseflan n lɛid tameẓyant.
Deg kra n lweqt kan, mmezlen 6 n yizgaren ara ferqen ɣef 730 n tesɣar n yimezdaɣ akk n taddart n Tala Ɛetman.
Akken tenniḍ d tafrara, yettwagzem uksum, yettwahegga d imuren.
Uqbel ad yettwafreq, teḍra-d yiwet n tsirit i yettuneḥsaben ɣer Leqbayel d asekkir n uɛeddi (Rite de passage), seg umured ɣer tikli: Yal agrud amecṭuh yebdan ileḥḥu, yewwi-d ɣef yimawlan-is ad t-skecmen ɣer lewziɛa.
Ad t-id-awin ass-nni i wakken ad yawi iɣes.
Qqaren-d akken ad yiǧǧhid u diɣ ad t-issinen Wat taddart.
Deg leɛwayed n Leqbayel diɣ, ttilin-d sin-nniḍen n yisekkiren.
Asekkir wis sin mi ara iɛeddi uqcic ɣer tnubi (enfance à l’adolescence).
Dagi, ad t-awin ɣer ssuq i wakken ad t-issinen yimeddukkal n yimawlan-is akked yiderman-nniḍen.
Syin, ad d-yaɣ aqerru n uzger.
Qqaren d asfillet i wakken ad yuɣal d aqerru n uxxam, d amasay.
Ma d asekkir wis tlata, aneggaru, d mi ara yaweḍ ɣer 18 n yiseggasen deg leɛmer-is, syin ad yekcem ɣer tejmaɛt n taddart, ad yuɣal d yiwen gar yiɛeggalen-is.
“Akka, yerna llan kra i yebɣan ad aɣ-beddlen leɛwayed-nneɣ.
Bɣan ad aɣ-beddlen laṣel akked yiẓuran.
Cwi kan ur zmiren ara”, i d-yenna yiwen umɣar.
Mbeɛd mi d-d-yella lbiɛ n yigelman, ttuferqent tesɣar s lferḥ d unecreḥ gar yimezdaɣ am yigellil am win yesɛan.
« Ass-a, cerken akk yimezdaɣ n taddart yiwen n yimensi.
Ha-t-an yiwen yiswi gar yiswiyen n lewziɛ.
Nekni, ur nettḥewwis ma d leḥlal neɣ d leḥram.
Neẓra kan dayen yelhan », i iwala umɣar-nniḍen.
Iḍ n wass-nni, tella-d tmeɣra tameqqrant anda yecna ucennay mechuren n taddart n Tala Ɛetman, Youcef Guerbas.
