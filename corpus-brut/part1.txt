YOUCEF AOUCHICHE, aselway n useqqamu aɣerfan n lwilaya (APW) n Tizi Uzzu.
“Nessaram ad nesbed asebdad n ugellid Cacnaq di yennayer 2970 (2020)”.
Deg tdiwennit-agi, yemmeslay-d Mass Aouchiche Youcef, aselway n useqqamu aɣerfan n lwilaya n Tizi Uzzu iɛeddan s ukabar n FFS, ɣef yisenfaren s umata, annar n JSK, aseqdec n tmaziɣt akked temsal-nniḍen i icudden ɣer lwilaya….
Aɣmis n Yimaziɣen: Di tazwara, ma tzemmreḍ ad aɣ-d-tinniḍ anda yewweḍ usenfar n lebni n usebdad n ugellid Cacnaq di temdint n Tizi Uzzu ?
Aouchiche Youcef : Lecɣal n lebni n usebdad i ugellid Cacnaq, teddun akken iwata lḥal.
Aql-aɣ, nettraǧu ineǧǧaren n uẓru ad aɣ-d-awin tawleft taneggarut ara nesqerdec d umussnaw n umezruy aqbur, i yeṭṭafaren asenfar-agi s lqerb.
Nebɣa ad d-nessekcem imussnawen n umezruy, i wakken ur nttaɣlaḍ ara, imi ulac tugna n ugellid-agi Cacnaq.
Syin akkin, ad bdun axeddim n usebdad-agi s timmad-is, wagi si tama, ma si tama-nniḍen, d aheggi n umḍiq ideg ara yettwassbed.
Nebɣa ad t-nesbed deg tlemmast n temdint n Tizi Uzzu, nesumer-d yakan yiwen umḍiq n unekcum ɣer temdint n Tizi Uzzu.
Yetturaǧu, ad nessiweḍ ad nelli asebdad-agi i lmend n yennayer 2970 (2020), acu kan, nessaram ur d-nettmaggar ara aṭas n wuguren deg ubrid-nneɣ.
Nebɣa ad yili usebdad-agi s unedrez (bronze), i d aɣ-iḥettmen ad t-id-nexdem di Blida.
Ansi i d awen-id-tusa tikti n lebni-ines u d acu i d yiswi-nwen deffir waya ?
Tikti n lebni n usebdad-agi n Cacnaq, tusa-aɣ-d asmi nesfugel yennayer iɛeddan (2969).
Nemsefham s yiɛeggalen n Useqqamu Aɣerfan n Lwilaya n Tizi Uzzu i wakken ad nerr azal i tmagit-nneɣ d umezruy-nneɣ.
Yerna, Cacnaq, d yiwen i yuran isem-is s yisekkilen n wureɣ deg umezruy n Tefriqt n Ugafa.
Nekni nwala belli deg umaḍal merra ulac tama i d as-yerran tajmilt i urgaz-agi n umezruy, akken i tt-yuklal.
Lebni n usebdad-agi, ad yili si tama d tajmilt meqqren, ma si tama-nniḍen, d tuɣalin ɣer yiẓuran-nneɣ d tmagit-nneɣ.
I wakken, daɣen tasuta-agi tamirant neɣ tid i d-iteddun ad issinent amezruy-nsent.
Aseqqamu Aɣerfan n Lwilaya, yessaram s usebdad-agi ad as-yerr tajmilt i ugellid Cacnaq, imi d lwaǧeb fell-aɣ waya.
Ma yella wamek, ara aɣ-d-tefkeḍ taḥawact n wahil-ik, mbeɛd ugar n useggas d uzgen segmi telliḍ ɣef uqerru n Useqqamu Aɣerfan n Lwilaya n Tizi Uzzu ?
Mačči d ayen isehlen ad d-nefk taḥawact n useggas d uzgen, maca ad neɛreḍ ad d-nemmeslay ɣef yisenfaen imeqqranen.
Aṭas n lecɣal i yettwaxdamen deg lwilaya n Tizi Uzzu, annect-agi yusa-d s uxeddim amcarek gar Useqqamu Aɣerfan n Lwilaya d yiselwayen n tɣiwanin akked yimezdaɣ-nneɣ diɣen.
Mi d-nusa, nufa-d lwilya n Tizi Uzzu tesɛa agellel (retard) d ameqqran deg unegmu, ladɣa deg wayen yerzan tadamsa d tmettit.
Nekeni deg wahil-nneɣ, nefka azal d ameqqran i wannect-a, i wakken lwilaya-nneɣ ad tecbu tiyaḍ.
Nebda s tisedduyin d tdeblin iwulmen, nesnerna aḥric n temguri (industrie) d wayen-nniḍen.
Ad d-kemmleɣ deg yisenfaren yebdan yakan i nekemmel nekni gar-asen: Asenfar n unnar n 50 000 n yimukan n Boukhalfa, swayes i d-tesfaydi lwilaya-nneɣ, i yesɛan azal meqqren deg unegmu deg lwilya n Tizi Uzzu.
Asenfar wis sin, d abrid ameqqran ara yeqqnen gar temdint n Tizi Uzzu ɣer ubrid Asammar-Amalu.
Abrid-agi yettwaxdamen, amur txeddem lwilaya n Tizi Uzzu, imuhal-is imeqqranen kfan.
Qqimen-d kan sin n yixenduqen, imi i d-nemmugger uguren di lebni-nsen.
Iqqim-d diɣ amur-nniḍen d lwilaya n Tubiret i t-ixeddmen.
Yella diɣ usenfar n ubaraǧ n Souk N’tlata, anda lḥan aṭas lecɣal n lebni-ines, d ayen ara yessifsusen taɛekkumt n uɣbel n waman n tissit.
Nesɛa diɣ asenfar n usiwweḍ s ttawil n tilifirik, anda nessaweḍ ad nfak aḥric amenzu n usenfar-agi, yeqqnen akka tura gar temdint n Tizi Uzzu alamma taɣrest n Bouhinoun.
Ɣer tama n yisenfaren-agi, yecban aɛiwed d useggem n uḥric n tmerrit d yisensuyen, imi tuget deg-sen ilaq ad ttuɛawden, maca imi llan kra n yisensa ideg ur ddin ara lecɣal akken iwata neɣ ɛad ur bdin ara maḍi, yegla-d s wugur-nniḍen n lexṣaṣ n yimukan n lembat n medden.
Ula deg uḥric n lebni, aṭas i nuẓa ɣer sdat, ama deg wayen yeɛnan AADL, ama deg tnezduɣin timettiyin neɣ deg tmezduɣin LSP.
Ad d-rnuɣ ad d-fkeɣ taḥawact n uḥric n waddal d yilmeẓyen, imi segmi i d-nusa nessaweḍ ɣer 49 n yinurar n wurar, iwumi nefka ussan iɛeddan tadrimt, i d aɣ-d-isqamen s 94 n yimelyaren.
Nerna azal n 80 n yimelyaren i 10 n yinurar i yettwajerden yakan i wakken ad ttwaṣegmen sya ɣer sdat.
Am wakken diɣ i nefka 89 n yimelyaren i wakken ad d-neɣrem tixessarin swayes i d-tegla ccetwa-nni iɛeddan ladɣa di Tegzirt d tamiwin i tt-iqerben.
Nesebɣes diɣ aḥric n usefti di tfellaḥt, dɣa nessaweḍ ɣer umḍiq amenzu deg wakal aɣelnaw deg usafar n uyefki, dayen i d aɣ-yesferḥen aṭas.
Ula d lɣella n zzit uzemmur d tbexsisin akked tamemt, yettkemmil usefti deg-sent, i wakken ad nessiweḍ ɣer lebɣi-nneɣ.
Yettban-d wannect-agi, deg usebɣes n tddukkliwin d ssrabes n tfellaḥt, i iqeddcen deg lɣellat-agi.
Rnu ɣur-s, ddeqs n yisenfaren-nniḍen yettwaxdamen di lwilaya-nneɣ ɣer tama n leqdic n yal ass.
Gar yisenfaren imaynuten i d-tewwim, d tukci n tedrimt i tɣiwanin i wakken ad snernit aseqdec n tutlayt tamaziɣt.
Awal ɣef wannect-a ?
Nekni s yiɛeggalen n Useqqamu Aɣerfan n Lwilaya, negzem-itt di rray deg temsalt-agi n useqdec d usnerni n tutlayt n tmaziɣt.
Imi segmi tuɣal tmaziɣt d tutlayt taɣelnawt tunṣibt, nezwer s usemres-ines deg Useqqamu-a.
D nekni i d Aseqqamu amenzu i yuran tilwiḥin yakk yellan daxel n Useqqamu-agi-nneɣ s tmaziɣt, am wakken daɣen i d-nesuzur yis-s ayen nxeddem.
I wakken ad taf amkan i d as-ilaqen, ilaq ɣef tsedduyin d yimḍebbren d yimezdaɣ, ad yexdem ayen zemren, yal yiwen di taɣult-is i lmend n userni n tutlayt-agi-nneɣ.
I lmend n waya, Aseqqamu Aɣerfan n Lwilaya n Tizi Uzzu, yefka tadrimt n 40 n yimelyunen i yal taɣiwant, i wakken ad taru tilwiḥin n tnilwin s tmaziɣt.
Am wakken diɣ, i d-neslul arraz n tɣiwant ara yesqedcen tamaziɣt ugar deg wakal-is, yetturaǧu ad yettwafraq deg tmeɣra n usfuggel n yennayer 2070.
Arraz-agi, deg tezrigt-is tamezwarut, ad iḥaz aṭas n yiḥricen, yezmer ad yili n twennaḍ, yezmer ad yili d tedbelt neɣ ama deg useddu n tɣiwant n yal ass.
Taseqqamut n yidles n Useqqamu Aɣerfan n Lwilaya s tallelt n Ugezdu n tutlayt d yidles n tmaziɣt n tesdawit n Mulud Mɛemmeri, ṭṭafaren axeddimen-agi.
Ma d iswi-nneɣ deffir waya, d axlaq n temḥezwert gar 67 n tɣiwanin-nneɣ ɣer tama n usnerni n tutlayt tamaziɣt di tmetti.
Nessarem ad d-yawi warraz-agi ayen yelhan i lwilaya-nneɣ, am wakken i t-yewwi warraz n Aissat Rabah n tezdeg deg uḥric n useḥbiber ɣef twennaḍt.
Dɣa, ma tzemmreḍ ad aɣ-tefkeḍ taḥawact n tezrigt-agi tis sebɛa n temsizzelt Aissat Rabah n tezdeg ?
Mbeɛd mi tessuli tallit tamezwarut n ujerred akked tis snat n ujerred n tudrin yebɣan ad ttekkint deg tezrigt tis sebɛa n temsizzelt n tezdeg Aïssat Rabah, yessulin ussan-agi kan deffir-nneɣ, ugar n 70 n tuddar i ijerden iman-nsen i tezrigt-agi tamaynut.
Ulamma, yenqqes s waṭas wuṭṭun-agi ɣef useggas iɛeddan, i yewwḍen ɣer 102 n tuddar, maca nessaram ad tili xir n tezrigt iɛeddan.
Syin akkin deg wussan i d-iteddun, ad tebdu Tesmilt yettwakelfen s twennaḍt d tezdeg akked tdawsa deg Useqqamu-a tuffɣiwin-is ɣer tuddar yettekkan deg tezrigt-agi i usenqeḍ-nsent.
Nessaram ad nessiweḍ, ad d-nefk arrazen-agi, i lmend n usmekti n umulli n tmenɣiwt n Aissat Rabah, ass n 12 tuber i d-iteddun.
Ma deg wayen yerzan amaynut n tezrigt-agi tamaynut, neslul-d timḥezwert “Best off” n tezdeg, ara d-yilin gar 6 n tuddar i d-yufraren deg seddiset n tezrigin timenza.
Ma d iswi-ines d asebɣes n ukemmel n usizdeg n tuddar-agi, imi nekni nettḥettim ɣef tuddar i d-yufraren deg temsizzelt-agi.
Ma d arraz yettuɛeggnen i temḥezwert-agi, mazal ur d-iban ara wazal-is u yerna ur iban ara ma ala taddart tamenzut neɣ snat-nni timenza ara d-yufraren iwumi ara nmudd arraz.
Ɣer tama n waṭas n yilugan-nniḍen d usishel n ujerred.
Annar amaynut n terbaɛt n JSK, yeṭṭili-d ur d-ikeccem….
Annar-agi mačči kan n terbaɛt n JSK, d annar akk i terbuyaɛ n lwilaya n Tizi Uzzu, maca tarbaɛt n JSK iban belli deg-s ara tetturar.
Ilaq ad nekfu deg-s axeddim di tallit iqerben, imi neḥwaǧ mliḥ agerraw anaddal n 50 000 n yimukan.
Imi d ayen ara d yawin lfayda ama si tama n tdamsa ama si tama n usnerni n waddalen di lwilaya-nneɣ.
Ahat annar n ddabex, ad ifak sya ɣer taggara n useggas-agi n 2019, imi ur yeqqim ara aṭas deg-s, mazal kan tuẓẓut n leḥcic (gazon) d tririt n ugeryaj.
Ma d agerraw s timmad-is, wissen ma ad ifak sya ar amseddis amenzu n useggas n 2020.
Ilaq ad tɛelmem belli asenfar-agi yeḥbes imi txuṣ tedrimt, imi am wakken teẓram tikebbaniyin-agi ixedmen deg-s xeddment s usebru, syin ad ttwaxlaṣent, maca annect-agi yettɛeṭṭil, dayen i d-yeglan s uḥbas n uxeddim deg-s.
Awal-ik n taggara….
Deg wawal-iw n taggara, ad awen-iniɣ tanemmirt-nwen s tussda imi tefkam azal i uxeddim i ixeddem Useqqamu Aɣerfan n Lwilya n Tizi Uzzu.
Tanemmirt-nwen daɣen, imi tettekkam s tira-nwen deg uɣmis-agi deg usnerni n tutlayt tamaziɣt, imi diɣ tgam azal i tutlayt-agi s tukci n 4 n yisebtar s tmaziɣt deg uɣmis-nwen.
Awi yufan ass-a, iɣmisen akk n tmurt-nneɣ ad fken azal i tmeslayt-agi-nneɣ imi tura ulac win izermren ad d-yaf ssebba, segmi tuɣal d taɣelnawt tunṣibt.
Ɣer taggara, ad asen-nini i yimezdaɣ-nneɣ, aql-aɣ yid-wen, nettakkawen-d tameẓẓuɣt ddima, memli i d aɣ-teḥwaǧem nella.

Deg tikli-agi tis 20 n umussu aɣerfan, nezmer ad d-nini ihubb-d qeccuc meccuc ɣer temdinin n Tmurt n Leqbayel.
Aṭas n wid ur nleḥḥu ara yakan, meɛna ffɣen-d tikkelt-a s abrid.
Yal yiwen d acu n ssebba i t-id-ideggren.
“Nekk, ffɣeɣ-d imi yettwagdel wannay amaziɣ.
Lliɣ ur d-cligeɣ ara deg umussu-agi.
Ur umineɣ ara yis-s akk”, i aɣ-d-yenna yiwen n yilemẓi i irefden asenǧaq amaziɣ.
Aṭas i yellan akka am netta iwumi iban yiswi n tikli.
Llan, diɣ, wid i yugin tamḥeqqranit akked uneḥbus n yimdanen, ladɣa, am yizumal n teqrawla tazzayrit.
“Ulamek ara neqqim deg uxxam mi ara twaliḍ tukksa n sser ɣef yigrawliwen neɣ ɣef yimeɣnasen.
Ulamek akk ara nizmir ad nessusem ɣef wayen i iḍerrun yid-sen.
Ilaq ad newwet fell-asen i waken ad asen-d-serḥen seg lḥebs”, i d-yerna yiwen ubabat i ileḥḥun d twacult-is deg temdint n Tizi Uzzu anda i d-ttmuddun yimezdaɣ-is aman i wid iteddun deg uɣamac n uzal qayli.
Xas ulamma yal wa d acu n ssebba i t-id-yewwin, meɛna yesdukkel-iten yakk yiwen yiswi: Aneqluɛ n unagraw seg yiẓuran.
Aya-gi yettban-d deg yisuɣan akked tira n yimesbaniyen i yuggulen ɣef rrwaḥ n wid i yellan deg udabu ama zik ama tura.
Gar yisuɣan n yimesbaniyen i yeṭṭfen taqacuct deg tekliwin n Tizi Uzzu, ad naf taggalit-a i d-yettuɣalen yal tikkelt “Ad truḥem, ad truḥem.
Nekni, nenna-d ad truḥem, ad truḥem”.
Awalen am wigi, qqnen mliḥ ɣer limin n Leqbayel.
Ttaken-d icbi deg jmaɛliman.
Ihi, akken qqaren: “Ma nniɣ-ak jmaɛliman, aẓ-d ma zadent deg-k tissas”.
Deg tikli-agi, daɣen, iban-d d akken ur qqbilen ara yimesbaniyen askasi-nni i d-yessumer uselway n uwanek Bensalah imi mazal ssuturen rwaḥ-is akked wid yakk yellan deg udabu akka tura, yedda-d ula d Gaid Salaḥ.
Akken yebɣu yili lḥal, mazal teddunt tikliwin deg talwit deg Tmurt n Leqbayel.
Hocine Moula.
