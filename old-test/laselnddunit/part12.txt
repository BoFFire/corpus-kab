Tura ihi, yya ad nemɛahad, ad nexdem leɛqed ayen ara yilin d ccahed gar-i yid-k.
Awit-iyi-d gma-twen amecṭuḥ, ad tḥeqqeɣ belli d tidet i d-teqqarem, mačči d ixbaṛǧiyen i tellam.
Mi sen-kfan irden-nni i d-wwin si Maṣer, Yeɛqub inna i warraw-is: Uɣalet ad ɣ-d-taǧwem cwiṭ n lqut.
Abṛaham itlef, Hagaṛ akked Ismaɛil, Ṣara twala aqcic-nni i d-tesɛa Hagaṛ tamaṣrit i Abṛaham, ittaḍṣa.
Yeɛqub izzwer kra seg iqeddacen-is, iceggeɛ-iten ɣer gma-s Meẓyan, i yellan di tmurt n Siɛir, di Adum.
Wwin-t warraw-is ɣer tmurt n Kanɛan, meḍlen-t di lɣar yellan di tferka n Maxfila, i yequblen Mamri.
Adam inna-as: D tameṭṭut-agi i yi-d-tefkiḍ ad tili yid-i, i yi-d-ifkan si ṭṭejṛa-nni, dɣa ččiɣ.
Tnac n iseggasen i llan ddaw n uzaglu n Kadurlaɛumir, lameɛna aseggas wis tleṭṭac, kkren-d ɣur-s.
Heggi-yi-d lmakla akken i tt- ḥemmleɣ, tawiḍ-iyi-tt-id ad t-ččeɣ, akken ad k-barkeɣ uqbel ad mmteɣ.
Ṛaḥil akked Liya rrant-as: Ɛni yella kra n nnṣib neɣ n lweṛt i ɣ-d-iqqimen deg uxxam n baba-tneɣ?
Ttasen-d si mkul tamurt ɣer Maṣer iwakken ad aǧwen irden ɣef Yusef, axaṭer laẓ izad di tmura meṛṛa.
Yusef inna i ugdud: Ass-agi, uɣeɣ-ken i Ferɛun, s kenwi, s tferkiwin-nwen.
Ɣef wayagi, argaz ad yeǧǧ baba-s d yemma-s, ad ilseq ɣer tmeṭṭut-is, dɣa ad uɣalen d yiwet n lǧetta.
Abṛaham irra-yas: Awi sebɛa n tzamarin-agi sɣur-i, ad ilint d ttbut belli d nekk i yeɣzen lbir-agi.
S wakka, Iḥitiyen ḍemnen i Abṛaham taferka-nni akked lɣar yellan deg-s, ad t-ikseb d amkan n temḍelt.
Axaṭer Ameɣlal ikkes dderya i tlawin meṛṛa n twacult n Dda Ḥemmu ɣef ddemma n Ṣara, tameṭṭut n Abṛaham.
Ameqran-nni yestkellfen s tissit inṭeq ɣer Ferɛun, inna-as: Ass-agi, ad d-mmektiɣ ddnub-nni i xedmeɣ.
D Yusef i yeɛeggcen baba-s, atmaten-is akked twacult meṛṛa n baba-s, akken i llan iɛeggalen-nsen meṛṛa.
Mazal ur kfiɣ lehduṛ deg wul-iw, mi d-teffeɣ Ribika, tacmuxt ɣef tayet, tṣubb ɣer lɛinseṛ ad d-tagwem.
Nekk, d Illu n Bitil, anda akken i tesbeddeḍ aẓru d ccahed, tdehneḍ-t s zzit, yerna tɛuhdeḍ-iyi dinna.
Tameṭṭut twala belli ṭṭejṛa-nni telha i wučči, teɛǧeb i tmuɣli, yerna d tin izemren ad d-tefk tamusni.
Teḥka-as tadyant yeḍran, tenna-yas: Akli-nni Aɛibṛani i ɣ-d-tewwiḍ iɛna-yi-d, ibɣa ad ilɛeb fell-i.
Tistan-nni icemten, iwumi i d-iqqim kan ugwlim ɣef yiɣes, ččant sebɛa n testan-nni icebḥen, iqewwan.
A-t-an twalam s wallen-nwen, ula d gma Binyamin iwala s wallen-is, d nekk, s timmad-iw, i wen-d-iheddṛen.
Abṛaham iɛenna ɣer Yillu, dɣa Illu yesseḥla Dda Ḥemmu, tameṭṭut-is akked tqeddacin-is: uɣalent ttarwent.
Mi sḍefrent wulli zdat n iɛekkzan-nni, ttarwent-d imecṭuḥen iberqacen, yesɛan izaggen, yerna reqmen.
Ferɛun ifka lameṛ i iqeddacen-is, ad t-ǧǧen ad iṛuḥ netta akked tmeṭṭut-is d wayen akk yellan d ayla-s.
Abṛaham akked Luṭ mfaraqen Abṛaham yuɣal-d si Maṣer ɣer Nigib, netta akked tmeṭṭut-is d wayen akk yesɛa.
Tesɛa-d aqcic, tsemma-yas Cit, tenna: Illu yefka-yi-d dderya-nniḍen, iɣrem-iyi-d Habil, i yenɣa Kayin.
Mbeɛd talalit n Qiynan, Anuc iɛac tmen meyya u xemseṭṭac n iseggasen, yerna yesɛa-d arrac akked tullas.
D wigi i d tnac n warraw n Ismaɛil, mkul yiwen s yisem-is, akken i llan iderman-nsen akked tudrin-nsen.
Laban inuda aqiḍun n Yeɛqub, aqiḍun n Liya akked uqiḍun n snat n tqeddacin-nni, lameɛna ur yufi acemma.
Ma yella wayen i tufiḍ, sseknit-id dagi zdat n watmaten-ik d watmaten-iw, d nutni ara tt-ifrun gar-aneɣ.
"Atmaten-is ggugmen zdat-s, seg wakken i ten-iffeɣ leɛqel.
 Yusef inna i watmaten-is: Qerrbet-d ɣur-i!"
Ur ken-ittɣiḍi ara wayen ara n-teǧǧem dinna, axaṭer ayen akk i tesɛa tmurt d lemxeyyeṛ, ad yuɣal nwen.
Tamnaṭ i zedɣen wat Ismaɛil, teṭṭef si Ḥawila armi d Cur, i yequblen Maṣer, deg ubrid ittawin ɣer Acur.
Yusef di lḥebs Mbeɛd ayagi meṛṛa, tameṭṭut n ssid-is terfed allen-is ɣer Yusef, tenna-yas: Ṭṭeṣ yid-i!
Illu yenna: Lqaɛa ad d-tefk tizzegzewt, leḥcic s zerriɛa-s, ṭṭjuṛ n lfakya n mkul ṣṣenf, s zerriɛa-nsent.
Ameɣlal Illu yenna i uzrem: Imi i txedmeḍ ayagi, tettwaneɛleḍ gar lmal meṛṛa, gar lḥiwan meṛṛa n lexla.
Mbeɛd talalit n Yirid, Mahalalil iɛac tmen meyya utlatin n iseggwasen, yerna yesɛa-d arrac akked tullas.
Tameddit n wass, tuɣal-d tetbirt-nni ɣur-s, tewwi-d deg uqabub-is yiwen n yifer n tzemmurt d aleqqaq.
Tameṭṭut n Abṛaham, isem-is Ṣara, ma d tameṭṭut n Naḥuṛ, isem-is Milka, yelli-s n Aṛan, baba-s n Iska.
Ameɣlal isxeḍ-d Ferɛun akked wat wexxam-is s lmuṣibat timeqranin ɣef ddemma n Ṣara, tameṭṭut n Abṛaham.
Abṛaham iẓẓa ṭṭejṛa n umemmay di Bir Cibaɛ, indeh dinna s yisem n Umeɣlal, Il Ɛulam, yeɛni Illu n dayem.
Ass-nni dɣa, iqeddacen n Isḥaq usan-d ɣur-s, wwin-as-d lexbaṛ, nnan-as: Nufa aman di lbir-nni i neɣza!
Nutni rran-as: Iqeddacen-ik d atmaten, nella di tnac yid-nneɣ, d arraw n yiwen n urgaz n tmurt n Kanɛan.
Mi s-ikkes leḥzen i Yahuda, yuli netta d uṛfiq-is Ḥira n Ɛadulam, ɣer Timna, ɣer wid ittellsen lmal-is.
Yeɛqub iṛuḥ ɣer Fadan Aram, ɣer Laban, mmi-s n Batwil Arami, gma-s n Ribika, yemma-s n Yeɛqub akked Meẓyan.
Yuɣ-it fell-asen yiwen n Umaṣri yettusemman Futifaṛ, yellan d aneɣlaf n Ferɛun, d ameqran n lɛeskeṛ-is.
Mbeɛd talalit n Mahalalil, Qiynan iɛac tmen meyya u ṛebɛin n iseggasen, yerna yesɛa-d arrac akked tullas.
Mbeɛd talalit n Nuḥ, Limix iɛac xems meyya u xemsa u ttsɛin n iseggasen, yerna yesɛa-d arrac akked tullas.
